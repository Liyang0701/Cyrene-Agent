# Separate Product Brand from Character Display Name

Cyrene Agent remains the Product Brand for the application, tray, technical logs, configuration directories, and product-owned infrastructure, while character-facing windows, calls, accessibility text, avatars, and messaging identity use the Active Character’s Character Display Name. Character Packages cannot rename the product or its storage, so implementation must dynamicize genuine character references such as “昔涟” without mechanically replacing “Cyrene” where it names the application or internal types.
