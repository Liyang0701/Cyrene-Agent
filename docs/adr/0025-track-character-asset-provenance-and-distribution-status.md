# Track character asset provenance and distribution status

Every Character Package declares the source and Package Distribution Status of its Live2D, avatar, voice model, reference audio, stickers, and text assets. Built-in or exported packages require explicit redistribution permission, while an `unknown/local-only` package may be installed with a persistent warning but cannot be exported, published, or committed; the application records these claims without pretending to adjudicate copyright, allowing private Hoshino testing while protecting the public repository from restricted assets.
