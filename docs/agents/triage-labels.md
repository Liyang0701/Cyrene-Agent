# Triage Labels

The engineering skills use five canonical triage roles. This file maps those roles to the labels in this repository’s GitHub issue tracker.

| Label in mattpocock/skills | Label in our tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a canonical triage role, use the corresponding label from this table.

Edit the right-hand column if this repository later adopts a different label vocabulary.
